## Http请求方式

```html
GET 获取数据
POST 修改数据
PUT  保存数据
DELETE 删除数据
OPTION 询问服务器的摸种支持特性
HEAD 返回报文头
```



## 状态码

```
2xx ：成功的
4xx ：客户端错误
5xx ：服务器错误
```
